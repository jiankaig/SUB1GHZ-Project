Web_esd_opcua - For cc3200 to send ESD and LED reading to opcua server

Prerequisite
1. Python 3 (pip3)
2. Notepad ++ (code editing)
3. Require to download opcua and cryptography packages

#install opcua
run pip3 install opcua on command prompt

#install cryptography
pip3 install cryptography


There are 4  object tags
1. ESD_1 	: ESD status 
2. Board_NO 	: Slave node number 
3. LED_R	: Red LED value 
3. LED_G	: Green LED value
4. LED_B	: Blue LED value
